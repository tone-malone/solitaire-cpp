# solitaire-cpp
C++ Solitaire


g++ compile command: 



g++ main.cpp -o solitaire -lSDL2 -lSDL2_ttf -lSDL2_image -lSDL2_mixer



Run command: 

./solitaire

## Running Tests

A small test suite is provided for utility functions. To build and run it:

g++ tests/util_tests.cpp -o util_tests
./util_tests
